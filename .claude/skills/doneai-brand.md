# Done.ai Brand Knowledge Base

Complete reference for Done.ai's brand identity, standards, and strategic positioning. This skill ensures consistency and quality across all brand touchpoints.

## Strategic Foundation

### What Done.ai Is

Advanced, unified business infrastructure platform integrating CRM, Accounting, Fintech, Inventory, and operations into a single automated core.

**Core Value Proposition**: Shifting from "Feature" to "Promise" - selling the finality of the result, not the process of working.

**Key Advantage**: Making an incredibly complex technical platform feel simple, understandable, and authoritative.

### Target Audience

**"The Essentialist"** (working concept, not finalized)
- High-level decision-makers and enterprise leaders
- Demand high-signal/low-noise environments
- Seek absolute operational truth
- Value authority and clarity over marketing hype

### Mission (Under Development)

Potential direction: Closing the "operational gap" between departments where profit and efficiency are traditionally lost.

*Note: Strategic positioning is being refined. These elements guide direction but aren't set in stone.*

## Verbal Identity

### Core Principles (Established)

**Mandate**: Authority through Simplicity

**Tone**: 
- Concise, direct, whisper-quiet
- No marketing hype or shouty language
- Confident but not arrogant
- Expert but not academic

**Primary Focus**:
- Resolution (end state) over Process (work)
- "What's done" rather than "how we do it"
- Though we do need to explain when necessary

### Forbidden Language

These words/phrases represent the old, fluffy approach:
- "Work smarter, not harder"
- Generic SaaS speak like "streamline," "optimize," "boost"
- Overly salesy or hype-driven language
- Focus on features over outcomes

### Brand Evolution Context

**Before (Old Approach)**:
- "Work smarter, not harder"
- "The go-to platform to streamline operations and manage key functions efficiently"
- Generic, could apply to any competitor
- Marketing-heavy, feature-focused

**After (New Approach)**:
- "All in one place. Done."
- "Command your entire infrastructure from a single, unified interface"
- Sharp, authoritative, outcome-focused
- Whisper-quiet confidence

## Visual Identity (Desktop Specifications)

### Color Palette (Established)

**Canvas (Environment)**
- Pearl White (#FBFBFB) - Primary background, external communications
- Midnight Violet (#0B0425) - Primary dark color, text, high-impact sections
- Platinum (#F0F0F0) - Secondary surface for cards/UI separation

**Brand & Soul**
- Royal Violet (#6A00D5) - "Energy" color for glows, pulses, active states, small accents only
- Lavender (#F4EFFF) - Hover states, soft highlights

**Utility (Status)**
- Dark Emerald (#006A4E) - Success/Completed
- Autumn Ember (#B45309) - Warning/Needs Attention
- Red Ochre (#DE350B) - Error/Blocked

*Note: These colors are working well in current design but not absolutely locked. Open to refinement.*

### Typography (Desktop Established Specs)

**Font Family**: DM Sans (all weights)

**Size Hierarchy**:
- Hero headline: 40px
- Hero subheading/tagline: 17px
- Section headings: 32px
- Section labels & body text: 16px

*Note: Mobile specs not yet documented. These are desktop specifications.*

### UI Components & Layout

**Buttons (Established)**
- Padding: 20px (vertical) × 8px (horizontal)
- Style: Full pill shape
- Primary: Midnight Violet background, Pearl White text

**Cards & Containers**
- Border-radius rule: Outer section padding minus inner card padding
  - Example: If section has 40px padding and card has 24px padding, radius = 16px
- Prefer shadows over borders for definition ("levitation" principle)

**Spacing**
- Follow 8pt grid system (specifics to be documented as design evolves)

### Visual Elements

**The Unified Core (Sphere)**
- Current: White porcelain/matte black sphere (placeholder)
- Future: High-end 3D animation planned
- Symbolizes "Order from Chaos"

**Potential Ideas (Not Decided)**
- "Ink Spill" transition: Light → Dark as users move from website to app
- Symbolizes shift from "Discovery" to "Command"

## Quality Validation

### Copy Review Checklist

When reviewing or creating Done.ai copy, check:

**Tone Check**
- [ ] Sounds authoritative but whisper-quiet (not shouty)?
- [ ] Focuses on outcomes/resolution rather than process?
- [ ] Avoids generic SaaS marketing speak?
- [ ] Could this apply to a competitor? (If yes, it's too generic)

**Forbidden Language Check**
- [ ] No "work smarter, not harder"?
- [ ] No unnecessary hype words (boost, unleash, streamline for its own sake)?
- [ ] No feature-focused language without outcome connection?

**Brand Evolution Check**
- [ ] Does this feel more like the "After" (sharp, direct) than "Before" (fluffy)?

### Visual Review Checklist

**Color Usage**
- [ ] Royal Violet (#6A00D5) used only for accents/glows/active states?
- [ ] Not used as primary background or large text color?
- [ ] High contrast maintained (dark on light or light on dark)?

**Typography**
- [ ] Using DM Sans?
- [ ] Using established size hierarchy (40/17/32/16)?
- [ ] Maintaining readability and hierarchy?

**Layout & Components**
- [ ] Buttons using correct padding (20px × 8px)?
- [ ] Card radius follows outer-minus-inner padding rule?
- [ ] Clean, minimal aesthetic maintained?

## Working With This Brand

### For Content Creation

When creating any Done.ai content (web copy, social posts, emails):

1. **Start with outcome** - What's done, not how
2. **Keep it quiet** - Authority doesn't need to shout
3. **Be absolute** - "We have resolved" not "We help you resolve"
4. **Check against "Before" examples** - If it sounds like old Done.ai, rewrite

### For Design Work

When designing Done.ai assets:

1. **Use the established color palette** - Pearl White, Midnight Violet as foundation
2. **Follow typography specs** - 40/17/32/16 hierarchy
3. **Apply spacing rules** - 8pt grid, card radius formula
4. **Reference the "After" examples** - Match that level of sophistication

### For Brand Strategy

Current focus (as of January 2026):
- Transitioning away from vague/fluffy messaging
- Building high-authority, sharp positioning
- Establishing Done.ai as the "Command Center" for enterprise
- Internal clarity: Brand as North Star for engineering/sales teams

**Features Section**

*Before*: "Work smarter, not harder" / "The go-to platform to streamline operations"

## Key Reminders

1. **This is a living brand** - Some elements established (colors, typography), others evolving (strategic positioning)
2. **Authority through simplicity** - Core principle guides everything
3. **Resolution over process** - Always focus on what's accomplished, not how
4. **Whisper-quiet confidence** - Let the work speak, don't oversell
5. **High contrast, high quality** - Visual sophistication matches message authority
